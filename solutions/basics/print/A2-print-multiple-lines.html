Print Multiple Lines

Learn how to break your output into multiple lines.

Solution Code

main.cpp
#include <iostream>
using namespace std;

int main() {

    cout << "Line 1" << endl;
    cout << "Line 2"<<endl;
    cout<<"Line 3"<<endl;

    return 0;
}

Code Explanation

We can use multiple cout statements to print on different lines. Each endl ensures that the next output will start on a new line. Notice how spacing around the << operator doesn't affect the program's output.

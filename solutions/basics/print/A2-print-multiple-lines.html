Print Multiple Lines

Learn how to print text on separate lines using endl.

Solution

#include <iostream>
using namespace std;

int main() {

    cout << "Hello" << endl;
    cout << "World";

    return 0;
}

Explanation

endl moves the cursor to the next line. This helps organize the output clearly instead of printing everything on one line.

Output

Hello
World

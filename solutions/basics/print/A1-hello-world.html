A1 • Beginner

Hello World

Learn how to print your first message on the console screen using cout.

Solution

#include <iostream>
using namespace std;

int main() {

    cout << "Hello World";

    return 0;
}

Explanation

The cout statement sends the text inside double quotes to the console screen. This is usually the first program beginners write while learning programming.

Output

Hello World

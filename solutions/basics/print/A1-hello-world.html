A1 • Beginner

Hello World

Learn how to print your first message on the console screen using cout.

Solution Code

main.cpp
#include <iostream>
using namespace std;

int main() {

    cout << "Hello, World!";

    return 0;
}

Code Explanation

We use cout to print text to the screen. The text must be enclosed in double quotes. This program correctly includes the <iostream> library and uses the std namespace to make printing possible.

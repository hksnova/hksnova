Escape Characters

Learn how to print special symbols and format text.

Solution Code

main.cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Name:\tHKSNova" << endl;
    cout << "Role:\tDeveloper" << endl;
    return 0;
}

Code Explanation

Here we are using the \t escape character, which acts like pressing the "Tab" key on your keyboard. It adds horizontal spacing to align the output neatly into columns.
